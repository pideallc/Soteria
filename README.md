# Soteria

**AI-First Feminist Cybersecurity Ecosystem**

Soteria is a practical, sex-positive feminist cybersecurity platform that shifts the threat model from corporate perimeter defense to protecting people — their bodies, identities, and autonomy.

## Overview

This is the unified project website for Soteria, integrating multiple design perspectives into a single comprehensive experience:

- **Vision** — The goddess archetype and founding philosophy of communal digital care
- **Intelligence** — Comparative threat model analysis (Mainstream vs. Soteria) with interactive radar chart
- **Architecture** — Neural-Symbolic system design: adaptive AI detection paired with deterministic ethical guardrails
- **Protocol** — The Blind Key Protocol: live SHA-256 client-side hashing demo showing zero-knowledge privacy engineering
- **Operations** — Interactive scenario simulations (stalkerware, identity segregation, financial sovereignty, doxxing defense)
- **Pioneers** — The activist and research organizations (Chayn, Hackblossom, OII Reconfigure, APC) whose work grounds Soteria
- **The Vault** — Community signup with client-side email hashing and optional Firebase persistence

## Technical Stack

- **HTML5** single-file application
- **Tailwind CSS** (CDN) for styling
- **Chart.js** for data visualization (radar chart)
- **Web Crypto API** for real SHA-256 hashing in the browser
- **Firebase** (optional) for anonymous auth and Firestore persistence

## Running Locally

Open `index.html` in any modern browser. Firebase features are optional and degrade gracefully when no configuration is provided.
